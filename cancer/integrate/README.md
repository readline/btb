# btb/cancer/integrate

Cancer related tools.

---

## MuSicSmgSort.R

  * MuSicSmgSort.R [MuSic smg file]
  * Sort the MuSic output smg file with P-value CT then tot muts.
  * Takes only one argument: smg file path
  * Output to [smg path].sorted
